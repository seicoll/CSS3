animation: name duration timing-function delay interaction direction;

Una animació permet que un element canviï gradualment d'un estil a un altre.
En les transicions només podiem definir un estat inicial i us estat final de l'element.
En canvi amb les animacions podem indicar diferents estats pels qual passarà l'element.

Per a això, ha d'especificar varies coses:

  • name: nom de l'animació definida amb un @keyframes
  • duration: la durada de l'animació
  • delay: temps que ha de passar abans que comenci l'animació
  • interaction: nombre de vegades que s'ha de reproduir l'animació. Per exemple: 1 | infinite
  • direction: reverse | alternate

Exemple Animació 1

	#quadre1
	{
		animation: animacio 5s linear;	
	}
	
	@keyframes animacio {
		0% {
			top: 100px;
			left: 50px;
			background-color: green;
		}
		25% {
			top: 100px;
			left: 350px;
			background-color: red;
		}
		50% {
			top: 250px;
			left: 350px;
			background-color: blue;
		}
		75% {
			top: 250px;
			left: 50px;
			background-color: yellow;
		}
		100% {
			top: 100px;
			left: 50px;
			background-color: green;
		}
	}
