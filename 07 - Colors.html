Colors: RGBa HSLa

RGBa (red, green, blue, alpha)

HLSa (hue, saturation, lightness, alpha)

El valor alpha és un número entre 0 i 1que determina l'opacitat del color.

0.0 : fully transparent
1.0 : fully opaque.

Color RGB
Sense transparència
color:rgba(50,50,50,1);
TEXT
Color RGBa
Amb transperència (Alpha)
color:rgba(50,50,50,.5);
TEXT
Color HSL
Sense transparència
color: hsla(0, 5%, 5%, 1);
TEXT
Color HSLa
Amb transperència (Alpha)
color: hsla(0, 5%, 5%, .5);
TEXT
